# OMR Salespage — VOGGSMEDIA

Next.js 14 single-page salespage for the OMR Masterclass on **2026-05-05**.
Hero feature is the **TikTok Ad Analyzer** — upload an ad, get a confidence
score and structured findings powered by Claude.

## Quick start

```bash
pnpm install
cp .env.example .env.local      # defaults run in mock mode (no API keys needed)
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000).
